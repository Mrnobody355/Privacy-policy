# Privacy-policy

Privacybeleid voor [Bot Naam]

Laatst bijgewerkt: 15 maart 2026

Dit privacybeleid beschrijft welke gegevens worden verzameld en hoe deze worden gebruikt wanneer je onze Discord bot gebruikt.

1. Welke gegevens we verzamelen

Onze bot kan de volgende gegevens verzamelen om goed te functioneren:

Discord User ID

Discord Server ID

Channel ID

Berichten die nodig zijn voor bot-functies (bijvoorbeeld moderatie of logs)

Wij verzamelen geen persoonlijke informatie zoals wachtwoorden of e-mailadressen.

2. Hoe we gegevens gebruiken

De verzamelde gegevens worden alleen gebruikt voor:

Het uitvoeren van bot-commando’s

Het opslaan van serverinstellingen

Moderatie- en logfuncties

Het verbeteren van de bot

3. Opslag van gegevens

Gegevens worden alleen opgeslagen zolang dit nodig is voor de werking van de bot.
We nemen redelijke maatregelen om deze gegevens te beschermen.

4. Delen van gegevens

Wij verkopen, verhuren of delen gebruikersgegevens niet met derden, behalve wanneer dit wettelijk verplicht is.

5. Verwijderen van gegevens

Server eigenaren of gebruikers kunnen verzoeken om hun gegevens te laten verwijderen door contact op te nemen met de bot eigenaar.

6. Wijzigingen in dit beleid

Dit privacybeleid kan op elk moment worden aangepast. De nieuwste versie zal altijd beschikbaar zijn via de officiële pagina.

7. Contact

Voor vragen over dit privacybeleid kun je contact opnemen via de Discord support server of met de eigenaar van de bot.
